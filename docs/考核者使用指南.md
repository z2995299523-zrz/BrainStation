# 考核者使用指南

考核者（Examiner）负责**创建和管理综合考试**，查看所有学习者的考试成绩和薄弱环节分析。

---

## 1. 角色说明

- 考核者可以**创建、发布、管理考试**
- 考核者可以**查看所有考生的成绩和短板分析**
- 考核者**不能**管理用户权限、修改系统配置
- 如需考核者权限，联系管理员通过管理面板升级

---

## 2. 考试管理入口

登录后导航栏显示「📝 考试管理」，点击进入管理界面。

---

## 3. 创建考试

### AI 生成考试（推荐）

1. **选择学科**：数学或英语
2. **选择章节**：勾选要覆盖的章节（多选），AI 会综合所有选中章节的知识点出题
3. **设置参数**：

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 难度级别 | 简单 / 中等 / 混合 / 困难 | 混合 |
| 题目数量 | 5~50 题（滑块调节） | 15 |
| 时间限制 | 10~180 分钟 | 60 |
| 及格线 | 10%~100% | 60% |
| 考试标题 | 可选，不填则自动生成 | — |
| 考试描述 | 可选，简要说明考试范围 | — |

4. 点击 **「🤖 AI 生成考试」** → AI（DeepSeek）会根据选中章节的：
   - 主题概要 (topic_summary)
   - 核心要点 (key_insights)
   - 常见错误 (common_mistakes)
   - 出题模板 (question_templates)
   
   自动生成覆盖所有知识点的试题（选择题 + 填空题混合）

5. 生成成功后，考试状态为 **「草稿」**，学生暂时看不到。

### 后续可由管理员在「用户管理」页面手动创建考试（如有需要）。

---

## 4. 管理考试

每场考试卡片显示：

| 信息 | 说明 |
|------|------|
| 标题 | 考试名称 |
| 状态 | 草稿（黄色）/ 已发布（绿色）/ 已归档（灰色） |
| 科目 | 数学 📐 / 英语 🔤 |
| 题目数 / 时长 | 如 15 题 / 60 分钟 |
| 参加人数 | 已完成考试的人数 |

### 操作按钮

| 按钮 | 功能 |
|------|------|
| **📊 成绩** | 展开/收起该考试的成绩汇总（参加人数、平均分、及格线、每人得分明细） |
| **发布** | 将草稿发布，学习者即可在考试列表中看到并参加 |
| **删除** | 删除考试及所有相关记录（需确认） |

---

## 5. 查看成绩

点击「📊 成绩」展开成绩面板：

**汇总数据：**
- 参加人数
- 平均分
- 及格线

**每人明细：**
- 用户名
- 得分 %（绿色通过 ✅ / 红色未通过 ❌）
- 正确 / 总题数
- 完成时间

**学习者的成绩单包含（学习者端可见）：**
- 总分及是否通过
- 每个章节的正确率柱状图（红色 < 60%，黄色 60%~80%）
- 逐题回顾：题干、你的答案、正确答案、解析

考核者可据此发现学生的共性薄弱点，调整教学内容。

---

## 6. 考试生命周期

```
创建（草稿） → 发布 → 学习者参加 → 查看成绩 → 归档（可选）
```

- **草稿**：可随时修改、删除
- **已发布**：学习者可见可参加；可删除
- **已归档**：不再显示在考试列表，但记录保留

---

## 7. 出题质量建议

| 建议 | 说明 |
|------|------|
| **覆盖均衡** | 选择 3~8 个章节，AI 会尽量覆盖所有选中的知识点 |
| **难度适中** | 混合难度适合大多数场景；困难模式适合选拔性考试 |
| **题目适量** | 15~25 题适合一节考试课（~60 分钟），不宜过多 |
| **验证题目** | AI 生成后抽查试题质量，必要时重新生成 |
| **定期更新** | 定期创建新考试，避免学生背答案 |
