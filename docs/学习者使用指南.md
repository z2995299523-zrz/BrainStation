# 学习者使用指南

欢迎来到 **初中数英思维训练营**！本指南帮助你快速上手日常学习流程。

---

## 1. 注册与登录

打开应用 → 点击「注册」→ 输入用户名和密码 → 注册成功后自动登录。  
已有账号点击「登录」。**新注册用户默认为学习者角色。**

---

## 2. 首页仪表盘

登录后进入首页，你可以看到：

| 区域 | 内容 |
|------|------|
| **欢迎语** | 显示用户名和连续学习天数 🔥 |
| **统计卡片** | 已掌握章节数、综合正确率、累计练习题数、连续学习天数 |
| **数学卡片** | 圆形进度环、掌握进度、正确率、练习题数、「开始数学学习」按钮 |
| **英语卡片** | 同上，英语学科 |
| **知识树** | 每张卡片内展示该学科的知识结构，绿色=已掌握、蓝色=学习中、灰色=未解锁 |

点击 **「🚀 开始数学学习」** 或 **「🚀 开始英语学习」** 进入章节学习。

---

## 3. 五阶段学习流程

每个章节按 **5 个阶段** 依次推进：

### 阶段 ① 概念 (Concept)
- 先看「章节概览」了解本章知识点
- 逐页浏览各小节详细讲解
- 阅读完后点击「进入例题」

### 阶段 ② 例题 (Examples)
- 逐题查看典型例题
- 每道题可展开/收起解题步骤
- 浏览完毕后点击「进入练习」

### 阶段 ③ 练习 (Practice)
- **选择题**：点击选项即选中，再点「提交」
- **填空题**：输入答案，回车或点「提交」
- 提交后弹窗提示 ✅正确 或 ❌错误+正确答案
- 可点击「💬 AI 导师」针对当前题目提问
- 全部做完自动进入下一阶段

### 阶段 ④ 测试 (Test)
- 与练习类似，但每道题有 **难度标注**（难度 1~5）
- 系统会记录你的 SM-2 答题数据（EF、间隔、掌握度）
- 做完所有题点击「完成测试」→ 展示得分和错题回顾

### 阶段 ⑤ 总结 (Summary)
- 庆祝动画 + 测试得分百分比
- 本章关键知识点回顾
- 「💬 让 AI 帮我总结」→ 打开 AI 导师
- 「🚀 进入下一章」→ 继续学习

**阶段导航条**（顶部 Stepper）：绿色打勾 = 已完成，蓝色 = 当前位置，灰色 = 待完成。可以点击已完成的阶段回看。

---

## 4. AI 导师

学习过程中随时可以打开 AI 导师：

- 右上角「💬 AI 导师」按钮 → 从右侧滑入对话面板
- 快捷提问：「换个方式讲」「再出一道题」「我错在哪」「总结本章」
- 也可以自由输入任何问题
- 支持多轮对话，对话历史自动保存
- 按章节组织对话，可新建/切换/删除会话

---

## 5. 综合考试

点击导航栏「📋 综合考试」进入考试列表：

- 查看所有已发布的考试（标题、科目、题数、时长、难度、及格线）
- 点击「开始考试」→ 逐题作答（选择题+填空题混合）
- 顶部显示进度条，底部可跳转到任意题号
- 绿色 = 已答对，红色 = 已答错，灰色 = 未答
- 做完点击「提交考试」→ 查看成绩单
- 成绩单包含：分数、是否通过、薄弱章节分析、逐题回顾（你的答案 vs 正确答案 + 解析）
- 可「再次考试」重新作答

---

## 6. 知识树与进度

- 每个学科的章节按依赖关系组成树状结构
- 父章节掌握后方可解锁子章节（掌握度 ≥ 85%）
- 掌握度由 SM-2 算法综合 EF、复习次数、是否超期计算
- 学习过程中自动更新状态：locked → unlocked → learning → mastered

## 7. 小贴士

| 技巧 | 说明 |
|------|------|
| **定期复习** | SM-2 算法会根据你的表现安排最佳复习间隔 |
| **善用 AI** | 遇到不懂的概念、做错的题，随时问 AI 导师 |
| **参加综合考试** | 跨章节测试能暴露知识盲区，考后看薄弱分析 |
| **数学用蓝色，英语用绿色** | 界面色彩区分，不易混淆 |
